📖 Algorithm Information

Linear Search

Linear Search is the simplest searching algorithm. It sequentially checks each element of the array until it finds the target or reaches the end.

  • Time Complexity: O(n)
  • Space Complexity: O(1)
  • Best Case: O(1) - target is first element
  • Works on: Any array (sorted or unsorted)
